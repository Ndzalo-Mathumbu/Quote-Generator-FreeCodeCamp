# QuoteFlow

QuoteFlow is a React-based Random Quote Machine that displays inspirational quotes from an external API. Users can generate new quotes and enjoy a dynamically changing color theme with every quote.

## Features

* Fetches random quotes from an API
* Displays quote text and author
* Generate a new quote with a button click
* Random color theme generation
* Responsive design using Bootstrap
* Loading indicator while fetching quotes
* Built with React functional components and Hooks

## Technologies Used

* React
* JavaScript (ES6+)
* SCSS
* Bootstrap 5
* jQuery
* Vite
* API Ninjas Quotes API
